# Self-hosted font choices

- Inter (body), IBM Plex Mono (terminal text), Bebas Neue (Bell Sign stand-in), and Fraunces 800 (Cooper Black stand-in) are copied from `@fontsource/*` packages.
- All four families are distributed under the SIL Open Font License through Fontsource/Google Fonts.
- Bebas Neue substitutes for the custom Bell signage face; Fraunces gives the heavy rounded display character without bundling proprietary Cooper Black.
